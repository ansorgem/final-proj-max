
To use this program you must have the following packages installed on your computer:
bs4
requests
json
spotipy
sqlite3
matplotlib.pyplot
pandas

To get API keys for spotify/spotipy go to developer.spotify.com and set up an account. 
Once you have the account set up, create an app in your dashboard and set an URI redirect within the app settings.
Then, you confirm the URI redirect by copy/pasting the URL in the provided prompt on the first run of the program- it should run without issue after.
